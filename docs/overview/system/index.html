平台基本功能概述

登录、找回密码、身份管理、应用管理、访问控制等模块的基础说明。

登录​

输入账号密码，默认管理员账号：

admin/ldapauth@2024

支持第三方社交登录，如钉钉、飞书、企业微信、微信扫码登录，前提是在系统中配置好社交凭证。

找回密码​

通过短信验证码方式进行密码重置，短信网关支持阿里云

身份管理​

组织管理​

提供组织层面基础信息的增删改查的维护。

用户管理​

提供用户员工层面基础信息的增删改查的维护。

用户组​

提供用户组维度的增删改查的维护。用户属于分组内的一个维度。

系统默认存在两个组：

超级管理员组：具备系统管理所有权限

所有已认证用户组：认证后的用户具备的权限

用户组授权菜单​

提供用户组授权系统菜单功能，授权有的菜单用户可访问。无授权的菜单用户不可访问。

应用管理​

系统支持的协议应用，根据需要进行配置。

访问控制​

该模块主要针对组与应用的授权关系，组下面的用户具备应用的访问权限
