钉钉同步配置

登录系统，找到同步中心->钉钉

参数说明

名称必填备注
接口前缀是默认为https://oapi.dingtalk.com，如私有化部署，请填写对应的域名
组织根标识是默认根节点ID : 1
应用凭证是开放应用的key
应用秘钥是开放应用的secret
任务调度是设置自动同步频率，系统在时间范围内进行定时同步
状态是正常的配置会进行同步，禁用后无法同步

测试配置​

配置完成后，可进行配置验证是否成功。

立即同步​

立即执行任务同步

同步记录​

同步后产生同步记录，同步了那些数据以及同步是否成功。

登录进入钉钉开放平台，通过路径： 应用开发 ---> 创建应用 。

创建成功之后，我们就能拿到该应用的 Key 与 Secret：

拿到之后可以将对应的配置填写到配置中，然后就是权限申请，分别需要如下权限：
